Speech Recognition
==================
simple speech recognition and text to speech in python.

---
### Libraries:
```
pip install pyttsx3
pip install pyaudio
pip install SpeechRecognition
```
Or just install requirements
```
pip install -r requirements.txt
```

* Note: For macOS:
```
brew install portaudio
```

### Run
```
python main.py
```
